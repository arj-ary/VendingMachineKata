Completed Functionalities for the Vending Machine Keta
------------------------------------------------------

Power Up
	Displays Insert Coin When Powered Up
	Display's Inventory when powered up

Accept Coin
	Vending machine accepts various coins and displays accepted value {Quarters, Nickels and Dimes}
	Vending machine displays aggregate values when more than one coin is inserted.
 
Make Product Selection
	User inputs exact price of the product and see's INSERT COINS on the display
	User inputs more amount than what is required and vending machine return amount or change.

Display Sold out
	Machine displays SOLD OUT message when user tries to fetch an item whose quantity is 0
